# Colcheia Pontuada

Sistema de Gerenciamento de Partituras para Orquestras, Institutos e Conservatórios.

## Descrição

Colcheia Pontuada é um sistema desenvolvido para gerenciar projetos musicais de medio e grande porte, como orquestras sinfônicas, institutos musicais ou conservatórios. Ele controla o acesso a arquivos de partituras por roles específicas, organiza grupos musicais, e repertórios, para otimizar ensaios e apresentações.

O projeto foi criado como trabalho de POO e BD.

## Entidades/Classe criadas

- Grupo
- Formação
- Maestro
- Músico
- Repertório
- Partitura
- Peça
- Instrumento
